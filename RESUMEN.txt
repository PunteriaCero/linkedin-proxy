                        → Compara con processed_messages.json
                        → Envía a n8n webhook
                        → Previene duplicados

3. 💬 RESPUESTAS

🧠 SIMULACIÓN MENTAL DEL FLUJO (COMPLETADA):

   PASO 1: Startup (10:00:00)
   └─ FastAPI inicia en puerto 8000

   PASO 2: Configuración (10:01:00)
   └─ Usuario abre /admin
   └─ Ingresa li_at, JSESSIONID, webhook_url
   └─ Gateway limpia JSESSIONID automáticamente
   └─ Valida con LinkedIn (get_profile)
   └─ Guarda en config.json

   PASO 3: Sincronización (10:15:00)
   └─ n8n → POST /sync
   └─ Gateway obtiene conversaciones de LinkedIn
   └─ Compara con processed_messages.json
   └─ Identifica 3 mensajes NUEVOS
   └─ Transforma a formato n8n
   └─ Envía a webhook n8n
   └─ Actualiza processed_messages.json

   PASO 4: Rate Limit 429 (10:25:00)
   └─ LinkedIn devuelve 429 Too Many Requests
   └─ Intento 1: Esperar 1s → Reintento
   └─ Intento 2: Esperar 2s → Reintento (backoff exponencial)
   └─ Intento 3: Esperar 4s → Intento 4 Éxito ✓

   PASO 5: Respuesta desde n8n (10:30:00)
   └─ n8n → POST /reply
   └─ Gateway envía mensaje a conversación
   └─ LinkedIn responde 201 Created
   └─ Responde a n8n con confirmación

   PASO 6: Cookie Expirada (11:00:00)
   └─ Usuario intenta guardar cookies antiguas
   └─ Validación falla con 401
   └─ Gateway muestra: "Cookie expirada"
   └─ Usuario regenera cookies en LinkedIn

🚀 CÓMO USAR:

   INSTALACIÓN (si aún no está):
   $ cd linkedin-n8n-gateway
   $ python3 -m venv venv
   $ source venv/bin/activate
   $ pip install -r requirements.txt

   PRUEBAS:
   $ python3 test_gateway.py
   > Deberías ver: RESUMEN: 10 ✅ | 0 ❌

   EJECUTAR:
   $ python3 main.py
   > 🚀 Iniciando en http://localhost:8000
   > Dashboard: http://localhost:8000/admin

   O USANDO EL SCRIPT:
   $ ./start.sh
   > Ejecuta venv + tests + server automáticamente

   CONFIGURAR:
   1. Abre http://localhost:8000/admin
   2. Ingresa:
      - li_at (de LinkedIn DevTools > Cookies)
      - JSESSIONID (de LinkedIn DevTools > Cookies)
      - n8n_webhook_url (tu webhook en n8n)
   3. Haz click en "Guardar & Validar"
   4. ✓ Gateway valida con LinkedIn
   5. ✓ Configuración guardada

   SINCRONIZAR:
   En n8n, crea un webhook que haga:
   POST http://localhost:8000/sync
